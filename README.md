# 📊 Dropi Power BI Analytics Dashboard

## 📌 Descripción
Dashboard desarrollado en Power BI para analizar la operación de un negocio de dropshipping, incluyendo desempeño de productos, dropshippers, devoluciones y logística.

## 🎯 Objetivo
Optimizar la toma de decisiones mediante el análisis de:
- Tasa de entrega
- Devoluciones
- Rechazos logísticos
- Performance por dropshipper

## 🧠 Herramientas utilizadas
- Power BI
- DAX
- Modelado de datos
- Power Query

## 📊 Análisis incluidos
- Top productos vendidos
- Productos con mayor devolución
- Ranking de dropshippers
- Análisis de novedades (rechazos)
- Distribución geográfica
- Evolución temporal
